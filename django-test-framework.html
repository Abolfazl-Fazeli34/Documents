📗 جدول مستندات Django REST Framework

ردیف بخش مستندات توضیح مختصر لینک مستقیم
1 🚀 Quickstart Tutorial آموزش سریع و قدم‌به‌قدم برای ساخت یک API ساده با DRF؛ شامل نصب، تعریف Serializer، View، URL و اجرای اولین درخواست https://www.django-rest-framework.org/tutorial/quickstart/
2 📘 API Guide راهنمای جامع استفاده از APIهای DRF شامل مفاهیم کلیدی مثل ViewSetها، Routers، Serializers، مجوزها، احراز هویت و غیره https://www.django-rest-framework.org/api-guide/
3 📚 Serialization آموزش کامل نحوه تبدیل داده‌های مدل (QuerySet) به JSON و بالعکس با استفاده از Serializerها؛ شامل Fieldها و Validatorها https://www.django-rest-framework.org/api-guide/serializers/
4 🌐 Views توضیح انواع Viewها (APIView, GenericAPIView, Mixins, Generic Views, ViewSets) و نحوه کنترل رفتار درخواست‌ها و پاسخ‌ها https://www.django-rest-framework.org/api-guide/views/
5 🛣 Routers راهنمای ساخت مسیرهای خودکار (URL routing) برای ViewSetها با Routerهای مختلف (SimpleRouter, DefaultRouter و سفارشی) https://www.django-rest-framework.org/api-guide/routers/
6 🔐 Authentication معرفی انواع روش‌های احراز هویت (Basic, Session, Token, JWT, OAuth و غیره) و نحوه پیکربندی و استفاده از آنها در DRF https://www.django-rest-framework.org/api-guide/authentication/
7 🔒 Permissions آموزش سیستم مجوزدهی DRF برای کنترل دسترسی کاربران به APIها بر اساس قوانین مختلف (IsAuthenticated، IsAdmin و غیره) https://www.django-rest-framework.org/api-guide/permissions/
8 🛡 Throttling محدود کردن نرخ درخواست‌ها (Rate limiting) برای جلوگیری از سوءاستفاده و کنترل بار روی سرور https://www.django-rest-framework.org/api-guide/throttling/
9 💾 Filtering and Pagination فیلتر کردن داده‌ها (با FilterBackend, DjangoFilterBackend و...) و صفحه‌بندی نتایج API با روش‌های مختلف (LimitOffset, Cursor) https://www.django-rest-framework.org/api-guide/filtering/
10 📝 Testing نوشتن تست‌های واحد و یکپارچه برای APIها با استفاده از امکانات DRF مثل APIClient و تست درخواست/پاسخ https://www.django-rest-framework.org/api-guide/testing/
11 🔄 Versioning مدیریت و پشتیبانی از نسخه‌های مختلف API به صورت همزمان (URL, Header, QueryParam و غیره) https://www.django-rest-framework.org/api-guide/versioning/
12 📦 Pagination تقسیم نتایج پاسخ API به صفحات مختلف با استفاده از کلاس‌های صفحه‌بندی پیش‌فرض یا سفارشی https://www.django-rest-framework.org/api-guide/pagination/
13 🔌 ViewSets and Mixins نحوه استفاده از ViewSetها (ModelViewSet, ReadOnlyModelViewSet و...) و Mixinها برای ساخت APIهای کوتاه و منظم https://www.django-rest-framework.org/api-guide/viewsets/
14 ⚙️ Settings تنظیمات کلی DRF شامل پیکربندی احراز هویت، مجوزها، رندررها، پارسرها، صفحه‌بندی و سایر گزینه‌ها در فایل settings.py https://www.django-rest-framework.org/api-guide/settings/
15 🧩 Parsers and Renderers نحوه خواندن درخواست‌ها و نوشتن پاسخ‌ها در فرمت‌های مختلف (JSON, XML, YAML و ...) https://www.django-rest-framework.org/api-guide/parsers/
16 📜 Schemas and Documentation تولید خودکار مستندات API با استاندارد OpenAPI (Swagger) و ابزارهای متنوع مثل CoreAPI، Redoc https://www.django-rest-framework.org/api-guide/schemas/
17 🔧 Customizing راهنمای سفارشی‌سازی رفتار DRF برای نیازهای خاص مثل تعریف Field سفارشی، Permission سفارشی، ViewSet سفارشی و غیره https://www.django-rest-framework.org/api-guide/customizing-views/
18 🛠 Browsable API رابط کاربری گرافیکی و تعاملی برای APIها جهت تست سریع، ارسال درخواست و مشاهده پاسخ‌ها در مرورگر https://www.django-rest-framework.org/topics/browsable-api/
19 📖 Tutorials and Examples مجموعه آموزش‌های گام‌به‌گام و مثال‌های کاربردی برای درک بهتر مفاهیم مختلف DRF و نحوه استفاده عملی https://www.django-rest-framework.org/tutorial/
20 🧩 Third-party packages معرفی کتابخانه‌ها و بسته‌های جانبی کاربردی و محبوب برای توسعه امکانات DRF مانند DRF-YASG، DRF-Extensions و غیره https://www.django-rest-framework.org/community/third-party-packages/
