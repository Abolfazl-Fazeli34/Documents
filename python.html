📘 جدول کامل مستندات Python 3.13.5

ردیف بخش مستندات توضیح دقیق و کامل لینک مستقیم
1 📌 What’s New in Python 3.13 معرفی ویژگی‌ها، بهبودها و تغییرات جدید پایتون ۳.۱۳ نسبت به نسخه‌های قبلی؛ شامل امکانات جدید زبان، اصلاحات کتابخانه استاندارد و تغییرات رفتاری مهم. مناسب توسعه‌دهندگان حرفه‌ای و کسانی که می‌خواهند به‌روز باشند. https://docs.python.org/3.13/whatsnew/3.13.html
2 📘 Tutorial آموزش رسمی و جامع پایتون از پایه تا پیشرفته؛ شامل مباحث متنوعی مانند ساختار زبان، توابع، کنترل جریان، ساختار داده‌ها، ماژول‌ها، شی‌گرایی و برنامه‌نویسی کاربردی. بسیار مناسب مبتدیان و کسانی که می‌خواهند زبان را گام‌به‌گام یاد بگیرند. https://docs.python.org/3.13/tutorial/index.html
3 📙 Language Reference مرجع کامل و رسمی زبان پایتون؛ شامل توضیحات دقیق درباره نحو زبان، قواعد گرامری، نحوه اجرای کد، مدل داده‌ها و انواع داده‌های زبان؛ منبع اصلی برای برنامه‌نویسان حرفه‌ای که می‌خواهند با جزئیات فنی زبان آشنا شوند. https://docs.python.org/3.13/reference/index.html
4 📗 Standard Library مستندات جامع و کامل همه ماژول‌ها، پکیج‌ها و توابع کتابخانه استاندارد پایتون؛ از مدیریت فایل، شبکه، پردازش متن، تاریخ و زمان، تا ماژول‌های پیشرفته مثل threading، multiprocessing و asyncio. بخش حیاتی برای استفاده روزمره و توسعه نرم‌افزار. https://docs.python.org/3.13/library/index.html
5 📂 Global Module Index فهرست الفبایی تمام ماژول‌های استاندارد و غیر استاندارد با لینک مستقیم به هر مستندات؛ ابزاری سریع برای جستجو و دسترسی به ماژول‌های مورد نیاز. https://docs.python.org/3.13/py-modindex.html
6 🧰 Python HOWTOs راهنماهای گام‌به‌گام برای انجام کارهای خاص و رایج در پایتون؛ شامل موضوعاتی مثل کار با فایل‌ها، استفاده از argparse برای آرگومان‌های خط فرمان، مدیریت استثناء‌ها، کار با Unicode، امنیت و بهینه‌سازی کد. https://docs.python.org/3.13/howto/index.html
7 📚 Python FAQs مجموعه سوالات پرتکرار کاربران درباره زبان پایتون؛ پاسخ به مشکلات رایج، خطاهای معمول، نکات مهم در برنامه‌نویسی، بهترین شیوه‌ها و توضیحات مفید برای رفع اشکال و درک بهتر زبان. https://docs.python.org/3.13/faq/index.html
8 🧑‍💻 Installing Python Modules آموزش نصب پکیج‌ها و کتابخانه‌ها با ابزارهایی مثل pip و setuptools؛ نحوه کار با PyPI، مدیریت محیط‌های مجازی (virtualenv)، به‌روزرسانی بسته‌ها، رفع مشکلات نصب و نکات مرتبط با توسعه و توزیع بسته‌ها. https://docs.python.org/3.13/installing/index.html
9 📦 Distributing Python Modules راهنمای ساخت، بسته‌بندی و انتشار پکیج‌های پایتون؛ توضیح ساخت فایل setup.py، استفاده از ابزارهای ساخت مدرن مثل setuptools و wheel، انتشار در PyPI و نکات کلیدی برای انتشار پکیج به صورت حرفه‌ای و استاندارد. https://docs.python.org/3.13/distutils/index.html
10 🛠 Extending and Embedding نحوه توسعه پایتون با زبان‌های C و C++؛ آموزش نوشتن افزونه‌های C، جاسازی مفسر پایتون در برنامه‌های دیگر، استفاده از API پایتون برای گسترش عملکرد، مناسب برای توسعه‌دهندگان کتابخانه‌ها و برنامه‌های سطح پایین. https://docs.python.org/3.13/extending/index.html
11 🧪 Python/C API Reference مستندات کامل و تخصصی API برنامه‌نویسی C برای پایتون؛ شامل انواع داده‌ها، توابع مدیریت حافظه، نحوه ساخت اشیاء، مدیریت استثناء و تعامل کامل با مفسر پایتون. مناسب توسعه‌دهندگان حرفه‌ای و سازندگان افزونه‌های پایتون. https://docs.python.org/3.13/c-api/index.html
12 🗂 Using Python on Unix نکات و ویژگی‌های خاص استفاده و نصب پایتون در سیستم‌عامل‌های Unix و Linux؛ شامل تنظیم محیط، اسکریپت‌های شل، مجوزها، مسائل امنیتی و بهینه‌سازی. مناسب برنامه‌نویسان لینوکس و سرور. https://docs.python.org/3.13/using/unix.html
13 🪟 Using Python on Windows راهنمای نصب، پیکربندی و استفاده از پایتون روی ویندوز؛ شامل نکات حل مشکلات رایج، تنظیم PATH، کار با PowerShell، مجوزها و تفاوت‌های پلتفرم ویندوز. https://docs.python.org/3.13/using/windows.html
14 🍏 Using Python on Mac توضیح نصب و پیکربندی پایتون در macOS؛ مدیریت نسخه‌های مختلف، استفاده از Homebrew، تنظیمات محیط و نکات مخصوص سیستم عامل مک. https://docs.python.org/3.13/using/mac.html
15 📚 Glossary فرهنگ اصطلاحات مهم و کلیدی زبان پایتون؛ مانند interpreter، iterable، generator، iterator، decorator، closure و غیره؛ برای فهم دقیق‌تر مستندات و مفاهیم تخصصی زبان بسیار کاربردی است. https://docs.python.org/3.13/glossary.html
16 📋 Index فهرست کامل و الفبایی تمامی مفاهیم، توابع، کلاس‌ها و ماژول‌ها در مستندات پایتون؛ بهترین ابزار برای جستجوی سریع و دسترسی به هر بخش مستندات. https://docs.python.org/3.13/genindex.html
17 📜 PEP Index فهرست کامل PEP ها (Python Enhancement Proposals)؛ شامل استانداردها، پیشنهادات توسعه زبان، ویژگی‌های جدید و نحوه تصویب آن‌ها؛ بسیار مهم برای کسانی که می‌خواهند درباره آینده و استانداردهای پایتون اطلاعات تخصصی داشته باشند. https://peps.python.org/
18 📥 Download Docs امکان دانلود مستندات کامل پایتون به فرمت‌های مختلف PDF، HTML، EPUB برای مطالعه آفلاین و آرشیو شخصی. https://docs.python.org/3.13/download.html
19 🔧 Debugging and Profiling ابزارها و ماژول‌های مرتبط با دیباگ کردن و پروفایلینگ برنامه‌ها؛ شامل استفاده از pdb، cProfile و دیگر ابزارهای اندازه‌گیری عملکرد کد. https://docs.python.org/3.13/library/profile.html
20 🔄 Import System شرح سیستم import در پایتون؛ نحوه بارگذاری ماژول‌ها، بسته‌ها، مسیرهای جستجو و جزئیات فنی مربوط به این مکانیزم کلیدی. https://docs.python.org/3.13/reference/import.html
21 🧩 Asyncio مستندات کامل کتابخانه asyncio؛ پشتیبانی از برنامه‌نویسی ناهمزمان، ایجاد event loop، coroutineها، taskها، و سینک کردن عملیات‌ها. https://docs.python.org/3.13/library/asyncio.html
22 🧮 Built-in Functions مرجع توابع داخلی پایتون مانند print, len, map, filter, zip و بسیاری دیگر؛ همراه با توضیحات کامل کاربرد و نمونه‌ها. https://docs.python.org/3.13/library/functions.html
23 🧱 Data Types معرفی انواع داده‌های پایه‌ای و پیشرفته در پایتون مانند list، dict، set، tuple، str، int، float و کلاس‌های داده‌ای. https://docs.python.org/3.13/library/stdtypes.html
24 📦 Packaging Python Projects راهنمای کامل بسته‌بندی پروژه‌های پایتون، شامل best practices، ساخت wheel، استفاده از setuptools، twine و بارگذاری در PyPI. https://packaging.python.org/en/latest/
25 🔒 Security Considerations نکات و بهترین روش‌های امنیتی هنگام برنامه‌نویسی با پایتون، مدیریت داده‌ها، استفاده امن از ماژول‌ها و جلوگیری از آسیب‌پذیری‌ها. https://docs.python.org/3.13/library/security.html
